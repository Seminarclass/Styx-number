STYX NUMBER

Sytx number helps us to understand the relation between the nature of bondings in borane and it's structure.


It's the code system, which represent the overall valence bonding in boranes by four digit number. In this,

S = Number of B-H-B bonds (First digit)

T = Number of B-B-B bonds (Second digit)

Y = Number of B-B bonds (Third digit)

X = Number of BH2 bonds (Fourth digit)


Styx number for various boranes on the basis of structure described above as

B2H6 = 2002

B4H10 = 4012

B5H9 = 4120

B5H11 = 3203

B6H10 = 4220

B10H14 = 4620


Example for (B2H6 - Diborane6)

Now we using B2H6 structure to find styx code

S = B-H-B = 2

T = B-B-B = 0

Y = B-B = 0

S = BH2 = 2

STYX CODE (B2H6) = 2002
